# Currency Converter

A live currency converter desktop app built with Python and CustomTkinter.

## Features
- Real time exchange rates via ExchangeRate API
- 12 major currencies supported
- Clean dark UI
- Instant conversion results

## Built With
- Python
- CustomTkinter
- Requests
- ExchangeRate-API

## Setup
1. Install dependencies: `pip install customtkinter requests`
2. Create a `config.py` file in the same folder
3. Add your API key inside it: `API_KEY = "your_key_here"`
4. Get a free key at [exchangerate-api.com](https://exchangerate-api.com)
5. Run: `python Currency-Convert.py`
